# aichat-java-study

本目录用于“运营学习代码”的独立 Java 工程（仅学习用途），不影响现有的 `aichat-backend` 等模块。

## 目录结构（按学习模块组织）

- `src/main/java/com/aichat/study/`
  - `concurrency/`：并发编程（当前主线）
  - `collections/`：集合框架与常用数据结构
  - `io/`：IO/NIO
  - `jvm/`：JVM/类加载/GC 相关实验
  - `lang/`：语言基础与语法细节（泛型、注解等）
  - `patterns/`：常见设计模式与实现练习
  - `utils/`：小工具与公共代码（学习期间沉淀）
- `src/test/java/...`：对应单元测试（后续写代码时再补）

> 约定：现在先只搭结构，不写具体代码文件；后续每次新增练习时再在对应包下创建类。
