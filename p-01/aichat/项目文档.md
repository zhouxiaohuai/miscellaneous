## 1. 项目概述

### 1.1 项目简介
- **项目名称**：AI Chat 项目
- **项目描述**：一个基于人工智能的聊天应用，提供智能对话、信息查询等功能
- **项目目标**：打造一个功能完善、用户友好的AI聊天系统

### 1.2 技术栈
- **前端**：React.js, TypeScript, CSS, HTML
- **后端**：Python, springboot
- **数据库**：MySQL
- **AI模型**：deepseek API

## 2. 项目结构

- **项目结构**：
  - 项目分为三个子项目：
    - **前端项目是一个子项目**：负责用户界面的开发和维护 子项目名称：aichat-frontend
    - **后端项目分为两个子项目**：后端项目分为spring的服务端项目和python的deepseek API项目
        - **服务端**：springboot用来处理前端请求，与数据库交互，调用AI模型等 子项目名称：aichat-backend
        - **AI模型**：python的deepseek API用来处理自然语言对话和信息查询 子项目名称：aichat-ai

## 3. 功能模块

### 3.1 核心功能
    - **主要目的**：提供智能对话、信息查询等功能，满足用户需求。
        前端有一个类似聊天框的界面，用户可以在界输入问题，AI会返回回答。
        后端服务端会调用AI模型，将用户的问题转换为自然语言，再调用AI模型进行回答。
        后端服务端会将AI模型的回答返回给前端，前端会将回答显示在界面上。
