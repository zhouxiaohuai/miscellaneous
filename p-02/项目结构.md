## Redis 学习路线

| 阶段 | 内容 | 对应文件 |
|------|------|----------|
| 1. 基础 | String/List/Hash/Set/ZSet 操作 | RedisBasicService |
| 2. 缓存 | 旁路缓存、读写穿透、缓存穿透/击穿/雪崩 | RedisCacheService |
| 3. 分布式锁 | Redisson 实现分布式锁 | RedisLockService |
| 4. 限流 | 滑动窗口、令牌桶 | RedisRateLimitService |
| 5. 事务联动 | Redis + 数据库事务一致性 | TxRedisService |
