# 项目总结

## 项目概述

**项目名称**：全局朗读助手  
**项目类型**：Android 原生应用  
**开发语言**：Kotlin  
**目标用户**：不识字人群  
**主要功能**：全局文字朗读

### 2. 核心功能代码 ✅

#### 主页面 (MainActivity.kt)
- 权限状态检查和显示
- 悬浮窗权限引导
- 无障碍服务引导
- 华为设备后台保护引导
- 启动悬浮按钮服务

#### 无障碍服务 (MyAccessibilityService.kt)
- 监听屏幕点击事件
- 递归遍历节点提取文字
- 支持点击和长按事件
- 调用 TTS 朗读文字
- 广播接收器控制开关

#### 悬浮按钮服务 (FloatingButtonService.kt)
- 全局悬浮按钮显示
- 支持拖动和点击
- 开启/关闭朗读服务
- 前台服务保活
- 通知栏提示

#### TTS 管理器 (TtsManager.kt)
- TTS 引擎初始化
- 中文语音合成
- 语速和音调控制
- 朗读状态监听
- 资源释放

#### 工具类
- `AccessibilityUtils.kt`：无障碍服务状态检查
- `HuaweiUtils.kt`：华为设备检测和兼容

#### 广播接收器 (BootReceiver.kt)
- 开机自启功能

### 数据流

```
用户点击文字
    ↓
AccessibilityEvent 触发
    ↓
MyAccessibilityService 接收
    ↓
提取 AccessibilityNodeInfo
    ↓
递归遍历获取文字
    ↓
TtsManager.speak(text)
    ↓
TextToSpeech 引擎朗读
    ↓
用户听到语音
```

1. 安装 Android Studio Hedgehog (2023.1.1)+
2. 安装 JDK 17
3. 安装 Android SDK 34

### 编译步骤

```bash
# 1. 用 Android Studio 打开项目
# 2. 等待 Gradle 同步完成
# 3. 连接设备或启动模拟器
# 4. 点击 Run 按钮
```

### 生成 APK

```bash
# 方法 1：Android Studio 菜单
```

## 注意事项

### 开发注意

1. **无障碍服务**：需要用户手动开启，无法自动获取
2. **悬浮窗权限**：需要用户手动授予
3. **华为后台**：需要引导用户设置后台保护
4. **TTS 引擎**：不同设备可能不同，需要测试兼容性

### 发布注意

1. **签名**：使用正式签名打包
2. **混淆**：配置 ProGuard 规则
3. **测试**：在目标设备上充分测试
4. **文档**：提供清晰的使用说明

### 使用注意

1. **权限**：首次使用需设置所有权限
2. **华为设备**：需要额外设置后台保护
3. **系统版本**：需要 Android 7.0+
4. **TTS 引擎**：确保设备有可用的 TTS 引擎
