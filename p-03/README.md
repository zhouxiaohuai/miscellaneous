# 全局朗读助手

一款面向不识字人群的 Android 应用，能够读取屏幕上的任意文字并语音朗读。

## 功能特点

- ✅ **全局朗读**：能够读取手机上任何应用的文字内容
- ✅ **悬浮按钮**：全局悬浮按钮，随时开启/关闭朗读服务
- ✅ **简单易用**：安装后只需几步设置即可使用
- ✅ **华为兼容**：专门适配华为手机（包括鸿蒙系统）

## 系统要求

- Android 7.0 (API 24) 或更高版本
- 推荐 Android 10.0+ 以获得最佳体验

## 安装和使用

### 1. 安装应用

下载 APK 文件并安装到手机上。

### 2. 首次设置

打开应用后，按照提示完成以下设置：

#### 设置悬浮窗权限
- 点击「去设置」按钮
- 找到「全局朗读助手」
- 开启「显示在其他应用上层」

#### 开启无障碍服务
- 点击「去设置」按钮
- 找到「无障碍」或「辅助功能」
- 找到「全局朗读助手」
- 开启无障碍服务

#### 华为手机特别设置（仅华为设备）
- 打开「手机管家」
- 点击「启动管理」
- 找到「全局朗读助手」
- 关闭「自动管理」
- 手动开启：
  - 允许自启动
  - 允许关联启动
  - 允许后台活动

### 3. 开始使用

1. 点击「开始使用」按钮
2. 最小化应用（或按 Home 键）
3. 打开任意应用（微信、浏览器、短信等）
4. 点击屏幕上的文字
5. 应用会自动朗读该文字

### 4. 控制朗读

- **开启/关闭朗读**：点击悬浮按钮
- **拖动按钮**：长按并拖动悬浮按钮到任意位置

## 功能说明

### 悬浮按钮

- 🎤 **麦克风图标**：朗读服务已开启
- 🔇 **静音图标**：朗读服务已关闭
- 点击按钮可切换状态
- 长按并拖动可调整位置

### 朗读功能

- 自动识别点击位置的文字
- 支持中文朗读
- 朗读完整句子或段落

## 常见问题

### Q: 为什么点击文字没有朗读？

A: 请检查：
1. 无障碍服务是否已开启
2. 悬浮按钮是否显示为「开启」状态（麦克风图标）
3. 重新打开应用检查权限状态

### Q: 华为手机后台服务被杀怎么办？

A: 请按照华为手机特别设置步骤，开启后台保护：
1. 打开「手机管家」
2. 「启动管理」中关闭本应用的「自动管理」
3. 手动开启所有后台权限

### Q: 如何调整语速？

A: 当前版本使用默认语速，后续版本会添加语速调节功能。

### Q: 支持哪些语言？

A: 当前版本主要支持中文，后续版本会添加多语言支持。

## 技术架构

```
app/
├── MainActivity              # 权限引导页面
├── service/
│   ├── MyAccessibilityService # 无障碍服务（核心）
│   └── FloatingButtonService  # 悬浮按钮服务
├── tts/
│   └── TtsManager            # TTS 语音合成管理
├── utils/
│   ├── AccessibilityUtils    # 无障碍服务工具
│   └── HuaweiUtils           # 华为设备兼容工具
└── receiver/
    └── BootReceiver          # 开机自启广播接收器
```

## 开发说明

### 环境要求

- Android Studio Hedgehog (2023.1.1) 或更高版本
- JDK 17
- Android SDK 34

### 编译运行

1. 克隆或下载项目
2. 用 Android Studio 打开项目
3. 等待 Gradle 同步完成
4. 连接手机或启动模拟器
5. 点击 Run 运行应用

### 生成 APK

1. Android Studio 菜单：Build → Build Bundle(s) / APK(s) → Build APK(s)
2. 等待编译完成
3. APK 文件位于 `app/build/outputs/apk/debug/app-debug.apk`
